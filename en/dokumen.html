Quick Guide

Use this quick guide to jump start on using Nessie Accounting.

Using Demo File

You can use the demo data file to evaluate Nessie Accounting. Download the file here.
Copy the data file (MetroBikeWorks.bak) to an external storage such as usb flash disk, then launch Nessie Accounting.

On the first displayed page, click on Import From External Storage button and then choose tokonessie.bak from your external storage, then click Import button.

Now on the Data File panel, you will find the imported file name listed on this panel, choose it and then click Open File button.

On Login dialog fill in
User Name : admin
Password : 12345
click on Login

Once logged in, you can access all functionality of the application. At the top most titlebar you will find menu bar where you can access most functions, and on the bottom left you will see the User Name currently logged in, you can click on this name to access Logout and Exit function.

Refer to this documentation for complete guide to use this application, you can access this document from within the application from menu bar Help Documentation . Navigate through the available subjects from the navigation side bar. If the navigation bar is not showed, you can click on the button located on the top-left side to open the navigation pane.

If you want to create a new file instead using the then you can follow the instruction below to create a new data file and initate it for first use.

Create New File

On the first displayed page, click on Create New File button to create a new data file.

On Business Information page, fill in with information of your company or organization, those with red placeholder text are mandatory. Choose the country from the drowdown list, by default the currency field will be selected for the national currency of your selected country. You can change it by selecting differently from the currency dropdown list. Attention: Country and Currency is non changeable after you completed this step.

Continue to next page by clicking on

On Book Period page, fill in with your accounting book period, this is the date when you start recording your transaction for the fiscal year, and complete it with the corresponding end date. The common used fiscal period are January 1st and December 31st respectively.

Continue by click on

On Master User fill the data of the person that will have the role as master user to this data file. A master user has the highest access authorization to this data file and cannot be overridden by other. Warning : password is changeable within the application but is non recoverable when lost or forgotten, make sure to remember or safe it well.

Continue by click on

On Data File page fill in the field with the name for the datafile without any file extension, the name should be alphanumeric without any white space and minimum 3 characters long.

Click on Save button to save your data.

At this point your data will be saved to the harddisk where your application reside (that will be in your PC or Laptop in most cases). It will be kept in a special folder maintained specially for your application. Be aware that this folder and all data in it will be removed when you uninstall the application. You can copy the data to an external disk from within the application. Refer to Other Setup - Data File Management

Initial Setup

The first displayed page after saving your new data file will be the Initial Setup Dashboard. On this step you can customize the Chart of Account to better suit you organization requirement, then you also need to set the automatic accounts before you can activate the database file.

You can get to the Chart of Account page by following this direction : Initial Settings Accounts Setting Chart of Accounts

On Chart of Account page you will find list of accounts that are used to record your transaction value. The application comes with a simple chart of account that you can use with minimal change to accomodate simple accounting need, you might want to change this to better fit your organization need. You can add +, remove - and edit accounts in this chart.

Chart of Accounts is structured as follow :
Category Group Sub Group Account
Where each Account must be a member of a Sub Group, and Sub Group of a Group. You can alter your chart of account any way you like as long as it follows this hierarchy.

There also some type of account that must present in your chart of account for the app to function, they are the accounts that are set as automatic accounts. You can learn more about Chart of Account in Initial Setting Chart of Account

On Linked Accounts Page, by default the fields will be populated with account that set as automatic and you can leave it as is. But in some cases when you change the chart of account structure, you might find these fields left empty and you need to manually set it by choosing from the dropdown control. If the dropdown is empty, then you need to create an account entry in the Chart of Account that is of the required account type (you can check on the question mark to find which type is required by each automatic account). Save the new setting by clicking on Save button (by default automatic populated fields are automatically saved and the save button is disabled/greyed).

Automatic Account only set once and can not be changed once the database is activated.

Database Activation

Before activating the database, make sure that the value on credit and debit sides are the same, you can check this by opening the Trial Balance page. On menubar Initial Settings Accounts Setting Trial Balance Check

Once these values are equal and linked accounts are set, you can activate your database by clicking on Activate Accounts button.

If the initial setup is set correctly, you can confirm the activation process by clicking on Activate Accounts button. If this button is greyed, then refer to the error message showed and make the necessary correction.

When activation process is succesfull, the full menubar will be displayed and you can start using the application to record your transactions.
