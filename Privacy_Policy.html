Privacy Policy

Do you know, when using Nessie Accounting software you might need to enter some information either business, personal, confidential or public in nature. It can be something like names, addresses, phone number, login credentials, etc. Therefore we need to explain to you about how these information will be stored and used by the software.

How is information used

The information you supplied to the software will only be utilized for your interest, it will be used in printing document or other places where this information is suitable for use.

Storage and security of your information

All information that are collected by the software will wholly and entirely be stored in a storage media managed by the software inside the user machine, namely in a hard disk or other kind of media where the database reside.

Developer access to user information

All information stored in the storage media managed by the software will only be accessible by the user from within the software. The developer or any third party doesn't have any way to access this information directly or indirectly.

User access to his information

User can access the information collected and stored by the software from the user interfaces provided by the software. Some information like name and address that has relation to a transaction can not be removed from the database for data integrity reason. If necessary, a user with Master Level authorization can remove the database in its entirety or partially when applicable. He/She can copy data file from the machine to an external media.

Policy expiration

This privacy policy is applicable starting by the time the software is being installed to a user machine and ended when the software is removed or uninstalled from it.
