Panduan Cepat

Ikuti panduan singkat berikut ini untuk mulai menggunakan aplikasi Nessie Akunting dengan cepat.

Menggunakan File Demo

Anda dapat menggunagakan file demo untuk mengevaluasi aplikasi Nessie Akunting, unduh file data disini.
Salin file data (tokonessie.bak) ke media eksternal seperti usb flashdisk, kemudian buka aplikasi Nessie Akunting.

Pada tampilan layar pertama, klik pada tombol Impor Dari Media Eksternal kemudian pilih file tokonessie.bak dari media eksternal anda, kemudian klik tombol Impor.

Anda akan melihat file yang baru saja anda impor ditampilkan pada panel File Data, pilihlah file tersebut kemudian klik tombol Buka File.

Pada layar Login isikan dengan data sebagai berikut,
Nama Pengguna / User Name : admin
Kata Sandi / Password : admin
Kemudian klik tombol Login

Setelah berhasil login, anda dapat mengakses fungsi-fungsi yang tersedia pada aplikasi. Pada bagian atas anda akan menemukan bilah menu yang berisi fungsi-fungsi untuk menjalankan aplikasi ini, baik untuk mencatat transaksi, melihat laporan-laporan, maupun pengaturan-pengaturan.

Sedang pada bagian kiri bawah ditampilkan nama pengguna yang menggunakan aplikasi, bila anda mengklik pada nama pengguna ini, sebuah menu akan menampilkan pilihan untuk Logout atau Keluar dari aplikasi.

Untuk mendapatkan petunjuk lengkap penggunaan aplikasi, anda dapat merujuk pada dokumen ini, dokumen ini dapat pula anda akses langsung melalui aplikasi Nessie Akunting dengan memiih menu Bantuan Dokumentasi, anda dapat membuka bagian yang berisi topik yang anda butuhkan melalui bilah navigasi yang tersedia di sisi kiri ini. Apabila bilah navigasi tidak tampak, anda dapat membukanya dengan mengklik tombol yang terletak disisi kiri atas.

Membuat File Baru

Apabila anda ingin membuat file data anda sendiri anda dapat mengikuti instruksi di bawah untuk membuat dan menginisiasi file data anda.

Pada tampilan Halaman Awal, klik tombol Buat File Baru.

Pada halaman Data Perusahaan isikan data perusahaan anda, pilih negara dari daftar negara yang tersedia, pilih mata uang yang digunakan, secara default aplikasi akan mengisikan dengan mata uang nasional dari negara yang dipilih sebelumnya. Perhatian: Negara dan Mata Uang tidak dapat diubah setelah pengisian awal ini.
Setelah selesai mengisikan data perusahaan / organisasi lanjutkan ke halaman berikutnya dengan mengklik pada tombol

Pada halaman Periode Buku isikan dengan tanggal mulai dan berakhirnya pencatatan transaksi keuangan.
Kemudian lanjutkan dengan menekan tombol

Pada halaman Pengguna Master isikan dengan data pengguna yang akan menjadi master dari file data perusahaan ini
Pengguna Master adalah pengguna yang memiliki otorisasi tidak terbatas terhadap data file perusahaan ini.
Perhatian: kata sandi/password harus diingat atau disimpan baik, karena aplikasi tidak menyediakan cara untuk memulihkan kata sandi yang hilang atau terlupa.
Kemudian lanjutkan dengan menekan tombol

Pada halaman File Data isikan dengan nama file untuk menyimpan data dari perusahaan yang akan anda buat.
nama file tidak boleh dipisah dengan spasi, harus terdiri dari 3 karakter atau lebih, dan terdiri dari huruf dan angka saja
Kemudian klik tombol Simpan untuk menyimpan data perusahaan anda.
File data anda akan disimpan dalam harddisk system pada folder yang khusus digunakan oleh aplikasi Nessie Akunting. Folder dan data didalamnya akan terhapus pada saat anda melakukan uninstall aplikasi Nessie Akunting, namun anda dapat menyimpan file data anda dengan melakukan penyalinan pada menu Manajemen Data.

Pengaturan Awal

Halaman pertama yang ditampilkan selanjutnya adalah halaman Dashboard Pengaturan Awal Sebelum anda dapat memulai pencatatan transaksi keuangan, anda perlu melakukan beberapa pengaturan dan mengaktivasi database perusahaan anda.
Anda dapat memulai dengan mempelajari daftar rekening (coa), pilih menu
Pengaturan Awal Pengaturan Rekening Buku Besar Daftar Rekening

Pada Halaman Daftar Rekening ditampilkan daftar rekening sesuai template default yang dapat digunakan sebagai dasar untuk menyusun daftar rekening anda.
Rekening-rekening tersebut disusun menurut susunan hirarki :
Group Sub Group Rekening
Dimana setiap Rekening Detail harus merupakan bagian dari suatu Sub Group, dan Sub Group bagian dari suatu Group.
Untuk menjadi bagian dari suatu Sub Group, Rekening harus memiliki no. rek lebih besar dari no. rek. Sub Groupnya tetapi lebih kecil dari no. rek. Sub Group berikutnya.
Untuk penjelasan lengkap anda dapat melihat dokuman pada bagian Pengaturan Awal Rekening Buku Besar

Pada Halaman Rekening Otomatis anda perlu memilih rekening-rekening yang akan digunakan sebagai rekening otomatis. Rekening otomatis adalah rekening-rekening yang diisi nilainya oleh aplikasi secara otomatis sesuai dengan transaksi-transaksi yang berkaitan. Secara default rekening-rekening ini akan diisi oleh aplikasi pada saat anda membuka halaman pengaturan rekening otomatis. Tetapi apabila anda mengubah daftar rekening maka ada kemungkinan anda perlu mengisikan rekening otomatis secara manual.
Pengaturan rekening otomatis hanya dapat dilakukan sebelum database diaktivasi dan tidak dapat diubah lagi setelahnya.
Untuk penjelasan lengkap anda dapat melihat dokumen pada bagian Pengaturan Awal Rekening Otomatis

Aktivasi Database

Sebelum anda mengaktivasi file database anda, pastikan bahwa nilai rekening pada sisi debit sesuai dengan nilai pada sisi kredit. Anda dapat memeriksa nilai ini pada Halaman Neraca Percobaan.
Bila nilai-nilai tersebut telah sesuai, anda dapat mengaktivasi file database anda dengan mengklik pada tombol Aktifkan Rekening

Apabila pengaturan telah dilakukan dengan benar maka anda dapat mengaktivasi database dengan mengklik tombol Aktifkan. Apabila tombol ini tidak tersedia atau tidak dapat diklik, pelajari pesan kesalahan yang ditampilkan dan lakukan pembetulan sesuai yang diminta.

Setelah aktivasi file database berhasil dilakukan, anda akan melihat tampilan awal Halaman Dashboard dengan bilah menu yang menampilkan menu lengkap. Anda dapat mulai menggunakan aplikasi untuk mencatat transaksi-transaksi anda. Pelajari lebih lanjut dengan memilih topik-topik yang tersedia di bilah navigasi di sebelah kiri, atau klik tombol di sebelah kiri atas untuk membuka bilah navigasi.

Instalasi dan Pembaruan Aplikasi

Ikuti panduan dibawah ini petunjuk mengenai cara-cara melakukan instalasi, pembaruan maupun uninstall.

Menginstal Aplikasi Nessie Akunting

Aplikasi Nessie Akunting hanya dapat diinstal melalui Microsoft Store yang terdapat pada Windows 10. Untuk menginstal dari Miscrosoft Store, anda dapat mencari melalui kolom pencarian yang tersedia di sebelah kanan atas dengan mengisikan nama Nessie Akunting pada kolom yang tersedia dan sebuah daftar temuan akan menampilkan pilihan untuk membuka halaman aplikasi Nessie Akunting di Microsoft Store. Pilih dengan mengklik pada daftar yang ditampilkan dan anda akan tiba pada halaman Nessie Akunting di Microsoft Store.

Aplikasi Nessie Akunting hanya dapat diinstal pada Windows 10 minimal pada Build 1904, untuk melihat build OS anda ketikkan kata "tentang" atau "about" pada bilah pencarian di Windows 10 yang terletak di sebelah kiri bawah, ketik "tentang" atau "about" kemudian pilih "Tentang PC Anda" atau "About your PC", sebuah jendela berisi informasi mengenai versi Windows 10 akan ditampilkan, gulung kebawah sampai anda melihat kolom "Build OS", apabila versi yang ditampilkan adalah sebelum versi 1904, anda dapat memperbarui dahulu versi Windows 10 anda melalui menu pengaturan di Windows 10 pada bagian "Pembaruan dan Keamanan "

Klik instal untuk mulai instalasi aplikasi Nessie Akunting. Windows 10 akan secara otomatis melakukan instalasi pada perangkat anda, anda dapat menutup jendela Microsoft Store bila menghendaki. Sebuah notifikasi singkat di sebelah kanan bawah layar anda akan memberitahukan apabila aplikasi Nessie Akunting telah terinstal dan siap untuk digunakan.

(apabila tombol instal tidak aktif atau tidak tersedia, maka kemungkinan versi Windows anda tidak sesuai, lihat penjelasan diatas mengenai versi Windows 10)

Membeli Aplikasi Nessie Akunting

Apabila anda sedang menjalankan versi uji coba dari aplikasi Nessie Akunting, maka pada bagian atas jendela akan dapat anda temukan sebuah tombol Versi Uji Coba, anda dapat mengklik tombol ini untuk membeli versi penuh dari aplikasi Nessie Akunting.

Cara lainnya adalah melalui Microsoft Store, pada bilah cari ketik Nessie Akunting kemudian pilih aplikasi Nessie Akunting dari daftar pilihan. Pada halaman aplikasi Nessie Akunting, klik pada tombol Beli

Pembaruan Aplikasi Nessie Akunting

Pembaruan untuk aplikasi Nessie Akunting akan diberitahukan melalui sebuah notifikasi berupa tombol hijau bertuliskan Pembaruan Tersedia yang akan ditampilkan pada jendela aplikasi pada bagian atas sebaris dengan bilah menu. Pembaruan dapat dilakukan dengan mengklik tombol tersebut, atau dapat juga dilakukan melalui Microsoft Store pada bagian Unduh dan Perbarui yang dapat anda temukan dengan mengklik pada tombol ... yang dapat anda temukan di sebelah kanan atas.

Pembaruan akan berjalan secara otomatis begitu anda mengklik tombol pembaruan, apabila pembaruan berhasil maka sebuah notifikasi singkat akan ditampilkan pada bagian kanan bawah layar anda. Apabila anda sedang menggunakan aplikasi kemungkinan aplikasi akan tertutup dengan sendirinya dan membuka kembali dengan versi yang terbarukan.

Menghapus Instalasi Aplikasi Nessie Akunting

Untuk melakukan penghapusan instalasi Nessie Akunting, temukan ikon Nessie Akunting pada Menu Awal (Start Menu) Windows 10, kemudian klik kanan pada mouse anda, sebuah menu pop-up akan ditampilkan, pilih Hapus Instalan dari menu tersebut untuk menghapus instalasi Nessie Akunting.

Peringatan : penghapusan instalasi Nessie Akunting akan menghapus pula file data yang tersimpan pada folder aplikasi di komputer/perangkat anda. Bila anda ingin menyimpan file data anda, maka lakukan penyalinan data ke media eksternal melalui menu manajemen data yang terdapat pada aplikasi.

Rekening Buku Besar

Rekening buku besar adalah kumpulan rekening-rekening neraca dan laba rugi yang digunakan untuk menyortir dan meringkas informasi yang telah dicatat dalam jurnal baik itu jurnal umum, junal penjualan maupun jurnal pembelian. Rekening-rekening ini dibuat menjadi sebuah daftar rekening yang terbagi menjadi 5 kategori utama yang bersifat tetap dan ditentukan oleh aplikasi.

Daftar Rekening

Pada bila menu pilih Rekening -> Daftar Rekening anda akan melihat tampilan seperti gambar disamping. Bagian atas berisi nama-nama Kategori yang terbagi seperti dijelaskan tabel di bawah.

Tambah Rekening

Untuk menambahkan rekening pada daftar rekening klik tombol + yang terdapat di bilah sebelah kiri daftar rekening. Panel Tambah Rekening akan ditampilkan di sisi kanan dari daftar rekening.
Mulailah dengan mengisi nomor rekening baru, dengan ketentuan nomor rekening baru harus lebih besar dari no rek. induknya namun lebih kecil dari no. rekening berikutnya yang memiliki hirarki 1 tingkat diatas rekening baru yang anda buat. Setelah no. rek. diisi lanjutkan dengan Jenis Entri, pilihan jenis entri akan tersedia secara dinamis menyesuaikan dengan posisi no. rekening anda dalam daftar rekening. Kemudian isi kolom-kolom berikutnya, setelah semua kolom diisi maka tombol Simpan akan diaktifkan secara otomatis, dan anda dapat mengkliknya. Apabila tombol tersebut belum aktif, berarti ada kolom isian yang belum anda isi dengan benar.
Kolom saldo awal hanya tersedia pada fungsi Tambah Rekening sebelum database diaktivasi dan hanya detil Rekening saja yang memiliki kolom Saldo Awal

Ubah Data Rekening

Untuk mengubah data Grup, Sub Group atau Rekening, pilih rekening dari daftar rekening kemudian klik pada tombol , panel Ubah Data Rekening akan ditampilkan di sisi kanan daftar rekening. Anda dapat mengubah nama dari Grup, Sub Grup dan Rekening pada fungsi ini, khusus untuk tipe entri Rekening anda dapat pula mengubah Saldo Awal rekening apabila database belum diaktivasi, setelah aktivasi anda hanya dapat mengubah nama rekening saja. Setelah perubahan yang diperlukan anda lakukan, klik tombol Ubah untuk menyimpan perubahan yang anda lakukan.

Hapus Rekening

Untuk menghapus Grup, Sub Grup atau Rekening, pilih rekening dari daftar rekening kemudian klik pada tombol - yang ada di bilah sebelah kiri daftar rekening, kemudian panel Hapus Rekening akan ditampilkan di sisi kanan daftar rekening. Klik pada tombol Hapus untuk menghapus rekening tersebut. Untuk dapat menghapus suatu Grup atau Sub Grup, maka Grup atau Sub Grup itu tidak boleh memiliki rekening anak dibawahnya. Sedangkan untuk menghapus suatu detil Rekening, maka Rekening tersebut harus memilik Saldo Nilai 0, tidak digunakan sebagai Rekening Otomatis, dan apabila database telah diaktivasi, maka rekening bersangkutan belum pernah digunakan dalam suatu transaksi.

Neraca Percobaan Ringkas

Dalam tahap pengaturan awal, fungsi Neraca Percobaan Ringkas diperlukan untuk melihat apakah nilai-nilai yang terdapat pada sisi debit dan kredit dari daftar rekening yang anda buat telah sesuai. Nilai Total sisi debit harus sama dengan sisi kredit sebelum anda dapat mengaktivasi database anda. Neraca Percobaan Ringkas ini dapat anda temukan di menu
Pengaturan Awal Pengaturan Rekening Neraca Percobaan

Rekening Otomatis

Rekening Otomatis adalah rekening-rekening yang digunakan oleh aplikasi dan nilainya diubah secara otomatis oleh aplikasi. Anda tidak dapat mengakses secara langsung rekening-rekening ini melalui fungsi-fungsi yang ada di aplikasi.
Pengaturan rekening otomatis adalah pengaturan untuk menentukan rekening-rekening yang akan digunakan sebagai rekening otomatis sesuai dengan ketentuan yang dapat anda temukan di petunjuk mengenai daftar rekening. Pengaturan rekening otomatis ini hanya dapat dilakukan pada saat database belum diaktivasi saja.
Secara default pada saat anda membuka halaman Rekening Otomatis, aplikasi akan mengisi kolom-kolom dengan rekening-rekening yang paling sesuai berdasarkan Tipe Rekening maupun Nama Rekening. Apabila anda melakukan perubahan pada daftar rekening dan merubah nama dari rekening-rekening tersebut ada kemungkinan aplikasi tidak dapat melakukan pengisian otomatis ini, sehingga anda perlu melakukannya secara manual dari daftar pilihan yang tersedia.
Setelah Rekening Otomatis diisi atau diubah, klik Simpan

Pengaturan Persediaan

Persediaan pada aplikasi Nessie Akunting disusun dari 2 komponen utama, yaitu daftar Lokasi/Gudang dan daftar Barang dan Jasa. Lokasi/Gudang berisi informasi dari lokasi/gudang dimana persediaan tersebut disimpan, suatu organisasi/perusahaan dapat memiliki lebih dari 1 lokasi. Sedangkan barang dan jasa berisi informasi mengenai tiap jenis barang berdasar nomor SKU nya. Setelah anda membuat daftar lokasi/gudang dan daftar barang maka anda dapat menambahkan Persediaan, dimana tiap persediaan mewakili ketersediaan suatu barang (item jasa tidak memiliki entri persediaan) di suatu lokasi tertentu, sehingga untuk tiap sku barang dapat memiliki banyak persediaan, masing-masing di tiap lokasi yang berbeda. Selanjutnya untuk membuat daftar Lokasi/Gudang, Barang dan Jasa, Persediaan anda dapat mengikuti petunjuk di bawah.

Daftar Lokasi/Gudang

Untuk menambahkan lokasi, klik tombol +, panel Tambah Data Lokasi akan ditampilkan pada panel di sisi kiri.
Isikan dengan data lokasi dimana anda akan menyimpan persediaan anda. Klik Simpan Lokasi yang baru saja anda tambahkan akan muncul pada panel Data Lokasi.
Pada kolom paling kanan dalam daftar lokasi, anda akan melihat tombol-tombol untuk melihat informasi detil dari lokasi, untuk mengubah data, dan untuk menghapus data lokasi.
